# memory_allocation

## Поедание памяти с помощью calloc

1. Зайти в папку *one*
2. Скомпилировать программу с помощью команды ```gcc one.c -o one```
3. Запустить программу с помощью команды, где `n` - колличество мегабайт выделяемых блоков памяти```./one n```
